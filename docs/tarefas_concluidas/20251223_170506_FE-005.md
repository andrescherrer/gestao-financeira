### Organização

A estrutura segue as melhores práticas do Next.js 14:
- **App Router**: `app/` para rotas e layouts
- **Componentes**: `components/` para componentes reutilizáveis
- **Utilitários**: `lib/` para código compartilhado
- **Tipos**: `types/` para definições TypeScript
- **Estáticos**: `public/` para assets
