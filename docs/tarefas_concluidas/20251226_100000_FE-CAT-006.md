# FE-CAT-006: Criar componente de seleção de categoria (combobox)

**Status:** ✅ Completo

## Resumo

Criação do componente `CategorySelect` para seleção de categorias. O componente utiliza o Select do Shadcn UI (baseado em reka-ui) e integra com a store de categorias para exibir apenas categorias ativas.

## Implementação

### Arquivos Criados

- `frontend/src/components/CategorySelect.vue`

### Funcionalidades

1. **Componente Select**:
   - Baseado em `Select` do Shadcn UI
   - Placeholder configurável
   - Suporte a estado de erro visual
   - Suporte a estado disabled
   - Label opcional

2. **Integração com Store**:
   - Carrega categorias automaticamente ao montar
   - Filtra apenas categorias ativas
   - Atualiza quando categorias mudam

3. **Props**:
   - `modelValue`: Valor selecionado (v-model)
   - `placeholder`: Texto do placeholder
   - `label`: Label do grupo (opcional)
   - `showLabel`: Mostrar label (padrão: false)
   - `disabled`: Desabilitar select
   - `error`: Mostrar estado de erro

4. **Emits**:
   - `update:modelValue`: Atualiza valor selecionado

### Componentes Utilizados

- `Select`, `SelectContent`, `SelectGroup`, `SelectItem`, `SelectLabel`, `SelectTrigger`, `SelectValue`: Shadcn UI Select

### Características

- Carrega categorias apenas quando necessário
- Filtra automaticamente categorias inativas
- Tratamento correto de tipos TypeScript
- Integração com vee-validate
