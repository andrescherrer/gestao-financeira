### Endpoints

1. **POST /api/v1/auth/register**
   - Body: RegisterUserInput
   - Response: RegisterUserOutput (201 Created)
   - Erros: 400 (validação), 409 (email duplicado), 500 (erro interno)

2. **POST /api/v1/auth/login**
   - Body: LoginInput
   - Response: LoginOutput (200 OK)
   - Erros: 400 (validação), 401 (credenciais inválidas), 500 (erro interno)

### Handler Methods

- **`Register`** - Processa registro de usuário
- **`Login`** - Processa login de usuário
