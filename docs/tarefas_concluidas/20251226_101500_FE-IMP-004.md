# FE-IMP-004: Implementar ordenação em tabelas

**Data:** 2025-12-26 10:15:00  
**Sprint:** 2.5 - Melhorias Frontend  
**Status:** ✅ Completo

## Resumo

Implementação de ordenação clicável nas colunas da tabela de transações, permitindo ordenar por data, descrição, tipo e valor.

## Implementação

### Arquivos Modificados

- `frontend/src/components/TransactionTable.vue`

### Funcionalidades

1. **Ordenação por Coluna**:
   - **Data**: Ordena por data da transação
   - **Descrição**: Ordena alfabeticamente
   - **Tipo**: Ordena por tipo (INCOME/EXPENSE)
   - **Valor**: Ordena por valor numérico

2. **Estados de Ordenação**:
   - **Sem ordenação**: Ícone `ArrowUpDown` (neutro)
   - **Crescente**: Ícone `ArrowUp` (ascendente)
   - **Descrescente**: Ícone `ArrowDown` (descendente)
   - **Clicar novamente**: Remove ordenação

3. **Interface**:
   - Headers clicáveis com cursor pointer
   - Ícones indicando estado de ordenação
   - Visual claro da coluna ordenada
   - Ordenação aplicada após filtros e antes de paginação

4. **Lógica de Ordenação**:
   - Toggle: null → asc → desc → null
   - Ordenação aplicada via computed `sortedTransactions`
   - Mantém ordem quando não há ordenação ativa

### Características

- Ordenação client-side (aplicada após filtros)
- Feedback visual claro
- Múltiplas colunas ordenáveis
- Integração com filtros e paginação

Seguido padrão comum de tabelas ordenáveis, garantindo familiaridade para o usuário.
