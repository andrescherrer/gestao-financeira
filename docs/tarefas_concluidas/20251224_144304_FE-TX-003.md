## Resumo

Criação do componente reutilizável `TransactionTable` para exibir transações em formato de tabela, extraído da página de lista de transações.

## Arquivos Criados/Modificados

### 1. `frontend/src/components/TransactionTable.vue`
- Componente reutilizável para exibir tabela de transações
- Props: `transactions` (array de Transaction)
- Colunas da tabela:
  - Data (formatada em pt-BR)
  - Descrição (com truncamento e tooltip)
  - Tipo (badge colorido: verde para receita, vermelho para despesa)
  - Valor (formatado como moeda, colorido por tipo)
  - Conta (nome da conta via useAccountsStore)
  - Ações (link para detalhes)
- Estado vazio com ícone e mensagem
- Hover states e transições

### 2. `frontend/src/views/TransactionsView.vue`
- Refatorado para usar componente `TransactionTable`
- Removida tabela inline
- Mantida função `formatCurrency` para estatísticas (cards)

## Detalhes Técnicos

### Formatação
- **Moeda:** `Intl.NumberFormat` com locale `pt-BR` e currency `BRL`
- **Data:** `Intl.DateTimeFormat` com locale `pt-BR` (DD/MM/YYYY)

### Integração
- Usa `useAccountsStore` para buscar nomes de contas
- Recebe transações via props
- Link para detalhes usando Vue Router

### UX
- Truncamento de descrição longa com tooltip
- Cores diferenciadas por tipo (verde/vermelho)
- Hover states na linha da tabela
- Estado vazio quando não há transações
