### Arquivos Criados

1. **`backend/internal/transaction/domain/events/transaction_created.go`**
   - Evento de domínio `TransactionCreated`
   - Contém informações sobre a transação criada (account_id, type, amount, currency)
   - Estende `BaseDomainEvent` para reduzir boilerplate

### Características

- ✅ Evento específico com informações detalhadas da transação
- ✅ Inclui account_id para identificar qual conta será atualizada
- ✅ Inclui transaction_type (INCOME ou EXPENSE) para determinar se credita ou debita
- ✅ Inclui amount e currency para atualização do saldo
- ✅ Implementa interface `DomainEvent`

### Estrutura do Evento

```go
type TransactionCreated struct {
    events.BaseDomainEvent
    accountID       string
    transactionType string // INCOME or EXPENSE
    amount          int64  // Amount in cents
    currency        string
}
```

### Métodos

- `AccountID()` - Retorna o ID da conta associada
- `TransactionType()` - Retorna o tipo da transação (INCOME ou EXPENSE)
- `Amount()` - Retorna o valor em centavos
- `Currency()` - Retorna o código da moeda

### Integração

- ✅ Evento criado automaticamente na entidade `Transaction` ao criar nova transação
- ✅ Substitui o evento genérico `BaseDomainEvent` por evento específico
- ✅ Publicado via Event Bus quando transação é criada
