## Resumo da Implementação

**Tarefa:** FE-TX-006 - Criar formulário de transação (React Hook Form + Zod)

### Arquivos Criados

1. **`frontend/components/transactions/TransactionForm.tsx`**
   - Formulário completo para criar/editar transação
   - Validação com Zod v4
   - React Hook Form para gerenciamento de estado
   - Campos implementados:
     - Conta (Select): busca contas do usuário via `useAccounts`
     - Tipo (Select): INCOME (Receita) ou EXPENSE (Despesa)
     - Valor (Input number): obrigatório, deve ser positivo
     - Moeda (Select): BRL, USD, EUR
     - Descrição (Textarea): 3-500 caracteres, com contador
     - Data (Input date): formato YYYY-MM-DD, padrão é data atual
   - Estados de loading e erro
   - Validação de formulário:
     - Todos os campos obrigatórios
     - Valor deve ser positivo
     - Descrição entre 3 e 500 caracteres
   - Botões de submit e cancelar
   - Desabilita submit se não houver contas disponíveis
   - Link para criar conta se não houver contas

2. **`frontend/components/ui/textarea.tsx`**
   - Componente Textarea do shadcn/ui instalado
