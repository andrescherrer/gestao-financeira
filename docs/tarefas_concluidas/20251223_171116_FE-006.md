# FE-006 - Criar layout base (Header, Sidebar, Footer)

**Data:** 2025-12-23  
**Status:** ✅ Concluída

## Resumo da Implementação

**Tarefa:** FE-006 - Criar layout base (Header, Sidebar, Footer)

### Componentes Criados

1. **Header** (`components/layout/Header.tsx`)
   - Header fixo no topo da página
   - Logo/título "Gestão Financeira"
   - Navegação principal (Contas, Transações)
   - Botão de Login
   - Sticky header com backdrop blur
   - Responsivo com container

2. **Sidebar** (`components/layout/Sidebar.tsx`)
   - Sidebar fixa à esquerda
   - Navegação vertical com ícones
   - Destaque para rota ativa
   - Links: Dashboard, Contas, Transações, Relatórios
   - Usa `usePathname` para destacar rota ativa
   - Altura calculada dinamicamente (`calc(100vh-4rem)`)

3. **Footer** (`components/layout/Footer.tsx`)
   - Footer fixo na parte inferior
   - Copyright com ano dinâmico
   - Links para Sobre e Contato
   - Separadores visuais

4. **Index** (`components/layout/index.ts`)
   - Arquivo de exportação centralizada
   - Facilita imports dos componentes

### Componentes shadcn/ui Utilizados

- **Button**: Botões de navegação e ações
- **Separator**: Separadores visuais

### Integração no Layout

**`app/layout.tsx`**:
- Header no topo
- Main com flex-1 para ocupar espaço disponível
- Footer na parte inferior
- Estrutura flex column para layout vertical

**`app/page.tsx`**:
- Integração do Sidebar
- Margem esquerda (`ml-64`) para compensar sidebar fixa
- Conteúdo principal com padding

### Estrutura de Layout

```

```
