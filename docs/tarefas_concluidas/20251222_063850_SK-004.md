**Tarefa:** SK-004 - Implementar interface DomainEvent e BaseDomainEvent

### Arquivo Criado

- **`backend/internal/shared/domain/events/domain_event.go`** - Base para eventos de domínio:
  - Interface `DomainEvent` com métodos obrigatórios
  - Struct `BaseDomainEvent` para reduzir boilerplate
  - Geração automática de IDs de evento (UUID)

### Interface DomainEvent

Define os métodos obrigatórios para todos os eventos de domínio:
- `EventID()` - ID único do evento
- `EventType()` - Tipo/nome do evento
- `AggregateID()` - ID do agregado que gerou o evento
- `AggregateType()` - Tipo do agregado
- `OccurredAt()` - Timestamp do evento
- `Version()` - Versão do schema do evento

### BaseDomainEvent

Struct base que implementa a interface DomainEvent:
- Reduz boilerplate em eventos específicos
- Gera automaticamente:
  - EventID (UUID)
  - OccurredAt (timestamp atual)
  - Version (1 por padrão)
- Permite embedding em eventos específicos

### Características

- ✅ Interface clara para eventos de domínio
- ✅ Base implementation para reduzir código repetido
- ✅ Geração automática de IDs (UUID)
- ✅ Timestamps automáticos
- ✅ Versionamento de eventos
- ✅ Suporte para evolução de eventos

### Uso

- Eventos específicos podem embedder `BaseDomainEvent`
- Facilita criação de novos eventos de domínio
- Padroniza estrutura de eventos no sistema
