# FE-GEN-005 - Adicionar ARIA labels para acessibilidade

**Data:** 2025-12-27  
**Status:** ✅ Concluída

## Resumo da Implementação

**Tarefa:** FE-GEN-005 - Adicionar ARIA labels para acessibilidade

### Implementação

Adicionados ARIA labels e atributos de acessibilidade em componentes principais para melhorar a experiência de usuários com leitores de tela e outras tecnologias assistivas.

### Arquivos Modificados

1. **`frontend/src/components/layout/Header.vue`**
   - `aria-label` em botão de toggle de tema
   - `aria-pressed` para estado do botão
   - `aria-label` em botão de notificações
   - `aria-describedby` para badge de notificações
   - `aria-label` e `aria-expanded` em botão de menu mobile
   - `aria-controls` para referenciar sidebar
   - `aria-hidden="true"` em ícones decorativos

2. **`frontend/src/components/layout/Sidebar.vue`**
   - `id="sidebar"` para referência
   - `aria-hidden` baseado no estado de abertura
   - `aria-label` na navegação
   - `role="list"` na lista de navegação

3. **`frontend/src/components/TransactionTable.vue`**
   - `role="columnheader"` nos cabeçalhos de tabela
   - `aria-sort` para indicar ordenação
   - `aria-label` em botões de ordenação
   - `aria-hidden="true"` em ícones

4. **`frontend/src/components/AccountCard.vue`**
   - `role="button"` para indicar interatividade
   - `aria-label` descritivo com informações da conta
   - `tabindex="0"` para navegação por teclado
   - Suporte a Enter e Space para ativação

5. **`frontend/src/views/ReportsView.vue`**
   - `aria-label` em botões de exportação
   - `aria-hidden="true"` em ícones

### Atributos ARIA Implementados

1. **aria-label:**
   - Descrições textuais para elementos interativos
   - Informações contextuais importantes

2. **aria-hidden:**
   - Ícones decorativos marcados como ocultos
   - Evita leitura redundante por leitores de tela

3. **aria-expanded:**
   - Estado de elementos expansíveis (menu, sidebar)
   - Indica se está aberto ou fechado

4. **aria-controls:**
   - Relaciona controles com elementos controlados
   - Ex: botão de menu controla sidebar

5. **aria-describedby:**
   - Relaciona elementos com descrições adicionais
   - Ex: badge de notificação descrito por aria-describedby

6. **aria-sort:**
   - Indica ordenação em tabelas
   - Valores: none, ascending, descending

7. **role:**
   - Define papel semântico de elementos
   - Ex: button, columnheader, list

8. **tabindex:**
   - Controla ordem de navegação por teclado
   - tabindex="0" para elementos interativos

### Benefícios
