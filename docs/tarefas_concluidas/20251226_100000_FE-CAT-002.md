# FE-CAT-002: Criar página de lista de categorias (/categories)

**Data:** 2025-12-26 10:00:00  
**Sprint:** 2.4 - Módulo de Categorias - Frontend  
**Status:** ✅ Completo

## Resumo

Criação da página `CategoriesView` para listar todas as categorias do usuário. A página segue o mesmo padrão visual e funcional usado em `AccountsView`, com cards para cada categoria e estados de loading, erro e vazio.

## Implementação

### Arquivos Criados

- `frontend/src/views/CategoriesView.vue`

### Funcionalidades

1. **Listagem de Categorias**:
   - Grid responsivo com cards para cada categoria
   - Exibição de nome, descrição e status (Ativa/Inativa)
   - Badge indicando status da categoria
   - Data de criação formatada

2. **Estados da Interface**:
   - Loading: Spinner com mensagem
   - Erro: Card com mensagem de erro e botão de retry
   - Vazio: Mensagem e botão para criar primeira categoria
   - Sucesso: Grid com cards de categorias

3. **Navegação**:
   - Breadcrumbs
   - Botão "Nova Categoria" no header
   - Cards clicáveis que redirecionam para detalhes

### Componentes Utilizados

- `Layout`: Layout principal da aplicação
- `Breadcrumbs`: Navegação hierárquica
- `Card`: Cards para categorias
- `Button`: Botões de ação
- `Badge`: Status da categoria
- `Loader2`, `Plus`, `Tag`, `AlertCircle`: Ícones Lucide

## Padrão Seguido

Seguido o mesmo padrão usado em `AccountsView`, garantindo consistência visual e funcional na aplicação.
