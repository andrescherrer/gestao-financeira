### Estrutura do Evento

```go
type AccountCreated struct {
    events.BaseDomainEvent
    userID      string
    name        string
    accountType string
    currency    string
    context     string
}
```

### Métodos

- `UserID()` - Retorna o ID do usuário dono da conta
- `Name()` - Retorna o nome da conta
- `AccountType()` - Retorna o tipo da conta
- `Currency()` - Retorna o código da moeda
- `Context()` - Retorna o contexto da conta (PERSONAL/BUSINESS)

### Integração

- ✅ Evento criado automaticamente na entidade `Account` ao criar nova conta
- ✅ Publicado via Event Bus quando conta é criada
