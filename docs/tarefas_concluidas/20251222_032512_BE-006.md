- ✅ Timestamp em RFC3339Nano
- ✅ Campo "service" adicionado automaticamente
- ✅ Logger global configurado
- ✅ Output para stdout

### Configuração

- **Variável de ambiente**: `LOG_LEVEL` (default: info)
- **Formato**: JSON estruturado
- **Timestamp**: RFC3339Nano
- **Service name**: "gestao-financeira"

### Integração

- Inicializado em `main.go` antes de outros componentes
- Usado pelo middleware de logger do Fiber
- Disponível globalmente via `logger.GetLogger()`
