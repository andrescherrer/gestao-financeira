### Estrutura de Uso

**Importar Componente:**
```typescript
import Button from 'primevue/button'
import InputText from 'primevue/inputtext'
```

**Usar no Template:**
```vue
<template>
  <Button label="Clique aqui" icon="pi pi-check" />
  <InputText v-model="value" placeholder="Digite algo" />
</template>
```

**Ícones:**
- Usar classes `pi pi-{nome-do-icone}`
- Exemplo: `pi pi-check`, `pi pi-user`, `pi pi-home`

### Temas Disponíveis

### Diferenças da Implementação Anterior (Next.js/React)

**Antes (Next.js/React):**
- shadcn/ui - Biblioteca de componentes para React
- Configuração via `components.json`
- Componentes instalados individualmente
- Baseado em Radix UI

**Agora (Vue 3):**
- PrimeVue - Biblioteca completa para Vue 3
- Configuração centralizada no `main.ts`
- Todos os componentes disponíveis por padrão
- Baseado em design system próprio do PrimeVue
