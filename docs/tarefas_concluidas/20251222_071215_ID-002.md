- **`backend/internal/identity/domain/valueobjects/password_hash_test.go`** - Testes unitários

### Características

- ✅ Hash de senha com bcrypt
- ✅ Validação de senha (comprimento, complexidade)
- ✅ Verificação de senha (Compare)
- ✅ Custo configurável do bcrypt (default: 10)
- ✅ Limites de comprimento (8-72 caracteres)
- ✅ Métodos de acesso (Hash, String)
- ✅ Verificação de vazio (IsEmpty)

### Validações

- Comprimento mínimo: 8 caracteres
- Comprimento máximo: 72 caracteres (limite do bcrypt)
- Validação de complexidade (opcional)

### Funcionalidades

- **`NewPasswordHashFromPlain`** - Cria hash a partir de senha em texto plano
- **`NewPasswordHashFromHash`** - Cria a partir de hash existente
- **`Compare`** - Compara senha em texto plano com hash
- **`Hash`** - Retorna o hash bcrypt

### Segurança

- Uso de bcrypt para hash
- Custo padrão: 10 (balance entre segurança e performance)
- Não armazena senha em texto plano

### Testes

- Criação de hash a partir de senha
- Validação de senhas inválidas
- Verificação de senha correta e incorreta
- Casos de borda (vazio, muito curto, muito longo)
