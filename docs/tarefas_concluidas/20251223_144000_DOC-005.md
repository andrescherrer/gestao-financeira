#### Localização da Configuração

**Arquivo:** `backend/cmd/api/main.go`  
**Linha:** 150-151


1. **github.com/swaggo/fiber-swagger**

   - Importado como `fiberSwagger`

2. **gestao-financeira/backend/docs**
   - Pacote gerado pelo `swag init`
   - Contém a documentação Swagger em formato Go

Após a configuração, os seguintes endpoints ficam disponíveis:

- `GET /swagger/index.html` - Interface Swagger UI
- `GET /swagger/doc.json` - Especificação OpenAPI em JSON
- `GET /swagger/doc.yaml` - Especificação OpenAPI em YAML
- `GET /swagger/*` - Outros recursos estáticos do Swagger UI
