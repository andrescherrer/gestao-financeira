# CAT-008: Configurar rotas de categories

**Data:** 2025-12-26 09:41:34  
**Sprint:** 2.3 - Category Context - Backend  
**Status:** ✅ Completo

## Resumo

Configuração das rotas HTTP para o contexto de Category, incluindo middleware de autenticação e integração com o handler.

## Implementação

### Arquivos Criados

- `backend/internal/category/presentation/routes/category_routes.go`

### Arquivos Modificados

- `backend/cmd/api/main.go`

### Rotas Configuradas

```
/api/v1/categories
├── POST   /              → Create category
├── GET    /              → List categories
├── GET    /:id           → Get category by ID
├── PUT    /:id           → Update category
└── DELETE /:id          → Delete category
```

### Middleware

- **AuthMiddleware**: Aplicado a todas as rotas do grupo `/categories`
- Proteção JWT para todos os endpoints

### Integração no main.go

1. Inicialização do repositório: `GormCategoryRepository`
2. Inicialização dos use cases: Create, List, Get, Update, Delete
3. Inicialização do handler: `CategoryHandler`
4. Configuração das rotas: `SetupCategoryRoutes`

### Estrutura

```go
categories := router.Group("/categories")
categories.Use(middleware.AuthMiddleware(jwtService))
{
    categories.Post("/", categoryHandler.Create)
    categories.Get("/", categoryHandler.List)
    categories.Get("/:id", categoryHandler.Get)
    categories.Put("/:id", categoryHandler.Update)
    categories.Delete("/:id", categoryHandler.Delete)
}
```
