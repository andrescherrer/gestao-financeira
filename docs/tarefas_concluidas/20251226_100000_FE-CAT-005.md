# FE-CAT-005: Adicionar seleção de categoria no formulário de transação

**Data:** 2025-12-26 10:00:00  
**Sprint:** 2.4 - Módulo de Categorias - Frontend  
**Status:** ✅ Completo

## Resumo

Adição do campo de seleção de categoria no formulário de transação (`TransactionForm`). O campo é opcional e permite ao usuário associar uma categoria a uma transação.

## Implementação

### Arquivos Modificados

- `frontend/src/components/TransactionForm.vue`
- `frontend/src/validations/transaction.ts`

### Funcionalidades

1. **Campo de Categoria**:
   - Campo opcional no formulário de transação
   - Posicionado entre "Moeda" e "Descrição"
   - Utiliza o componente `CategorySelect`
   - Validação UUID quando preenchido

2. **Validação**:
   - Campo opcional (não obrigatório)
   - Se preenchido, deve ser um UUID válido
   - Integrado com schema Zod existente

3. **Integração**:
   - Carrega categorias ativas automaticamente
   - Usa `useCategoriesStore` para buscar categorias
   - Mantém valor selecionado durante edição

### Nota Importante

O campo `category_id` foi adicionado ao formulário, mas o backend ainda não suporta esse campo nas transações. Quando o backend for atualizado para incluir `category_id` na entidade `Transaction`, a integração estará completa.

## Padrão Seguido

Seguido o mesmo padrão usado nos outros campos do formulário de transação, garantindo consistência na interface.
