# FE-IMP-006: Criar componente Dialog de confirmação

**Data:** 2025-12-26 10:15:00  
**Sprint:** 2.5 - Melhorias Frontend  
**Status:** ✅ Completo

## Resumo

Criação do componente `ConfirmDialog` reutilizável para confirmações de ações destrutivas ou importantes. O componente utiliza o Dialog do Shadcn Vue.

## Implementação

### Arquivos Criados

- `frontend/src/components/ConfirmDialog.vue`

### Arquivos Modificados

- `frontend/src/components/ui/dialog/` (instalado via shadcn-vue)

### Funcionalidades

1. **Componente ConfirmDialog**:
   - Props configuráveis:
     - `open`: Controla visibilidade
     - `title`: Título do diálogo
     - `description`: Descrição opcional
     - `confirmLabel`: Texto do botão de confirmação
     - `cancelLabel`: Texto do botão de cancelamento
     - `confirmVariant`: Variante do botão (default/destructive)
     - `isLoading`: Estado de loading
   - Emits: `update:open`, `confirm`, `cancel`

2. **Uso**:
   - Substitui `confirm()` nativo do navegador
   - Melhor UX e consistência visual
   - Suporte a loading state
   - Variante destructive para ações perigosas

3. **Integração**:
   - Preparado para uso em `CategoryDetailsView` (deleção)
   - Pode ser usado em outras views que precisam de confirmação

### Observação

Por enquanto, `CategoryDetailsView` ainda usa `confirm()` nativo, mas o componente `ConfirmDialog` está pronto para substituição quando necessário.

## Padrão Seguido

Seguido padrão de componentes de diálogo do Shadcn Vue, garantindo consistência visual.
