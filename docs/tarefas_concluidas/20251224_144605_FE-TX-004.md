## Detalhes Técnicos

### Formatação
- **Moeda:** `Intl.NumberFormat` com locale `pt-BR` e currency dinâmico
- **Data:** `Intl.DateTimeFormat` com locale `pt-BR` (DD/MM/YYYY)
- **Data/Hora:** `Intl.DateTimeFormat` com locale `pt-BR` (DD/MM/YYYY HH:mm)

### Integração
- Carrega contas automaticamente se ainda não foram carregadas
- Busca transação do servidor ou usa da lista se disponível
- Usa `useAccountsStore` para buscar nome da conta

### Estados
- **Loading:** Spinner com mensagem
- **Error:** Card vermelho com mensagem e botões de retry/voltar
- **Not Found:** Card centralizado com ícone, mensagem e botão para voltar

### Otimização
- Verifica se transação já está na lista antes de buscar do servidor
- Watch no route params para recarregar ao mudar ID
