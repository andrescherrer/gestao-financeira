# FE-IMP-007: Criar componente EmptyState

**Data:** 2025-12-26 10:15:00  
**Sprint:** 2.5 - Melhorias Frontend  
**Status:** ✅ Completo

## Resumo

Criação do componente `EmptyState` reutilizável para exibir estados vazios de forma consistente em toda a aplicação.

## Implementação

### Arquivos Criados

- `frontend/src/components/EmptyState.vue`

### Arquivos Modificados

- `frontend/src/views/AccountsView.vue`
- `frontend/src/views/TransactionsView.vue`
- `frontend/src/views/CategoriesView.vue`

### Funcionalidades

1. **Componente EmptyState**:
   - Props:
     - `icon`: Componente de ícone (Lucide)
     - `title`: Título do estado vazio
     - `description`: Descrição do estado
     - `actionLabel`: Texto do botão de ação (opcional)
     - `actionTo`: Rota para ação (opcional)
     - `actionIcon`: Ícone do botão (opcional)
   - Slot `action` para ações customizadas

2. **Uso**:
   - Substitui cards de empty state duplicados
   - Consistência visual em todas as views
   - Facilita manutenção e customização

3. **Integração**:
   - Usado em `AccountsView` (sem contas)
   - Usado em `TransactionsView` (sem transações)
   - Usado em `CategoriesView` (sem categorias)

### Características

- Design consistente
- Fácil customização via props
- Suporte a ações opcionais
- Ícones configuráveis

## Padrão Seguido

Seguido padrão de componentes de empty state comuns, garantindo consistência visual e facilidade de manutenção.
